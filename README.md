# Hotel-Price-Prediction
Hotel Price Prediction is a machine learning project that predicts hotel room prices using factors like location, room type, ratings, amenities, season, demand, and reviews. Selenium was used for data collection, Pandas for preprocessing, and Scikit-learn for model building.
## Project Structure
-> eda.ipynb – Used for Exploratory Data Analysis to understand the dataset and identify patterns.

-> model.ipynb – Used for training the machine learning model and evaluating its performance.
## Main Features 
-> Data cleaning and preprocessing

-> Feature selection and engineering

-> Model training

-> Performance evaluation and prediction analysis
## Technologies Used
Python
Pandas
NumPy
Scikit-learn
## Execution
Run the project in Google Colab
Open eda.ipynb for analysis
Open model.ipynb for training and testing the model
